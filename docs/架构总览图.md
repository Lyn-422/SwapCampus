## 一次访问的完整路径

```mermaid
sequenceDiagram
    participant 🧑 as 你(浏览器)
    participant 🚪 as Nginx<br/>(门迎)
    participant 🍳 as Django<br/>(后厨)
    participant ⚡ as Redis<br/>(白板)
    participant 🗄️ as MySQL<br/>(账本)
    participant 📦 as MinIO<br/>(仓库)

    Note over 🧑,📦: 场景一：打开首页（最简单）
    🧑->>🚪: 打开 geleme.online
    🚪-->>🧑: 直接返回首页文件（不经过后厨）

    Note over 🧑,📦: 场景二：搜索"二手教材"
    🧑->>🚪: 搜索 教材
    🚪->>🍳: /api/products/?search=教材
    🍳->>⚡: 缓存里有吗？
    ⚡-->>🍳: 没有，去查账本
    🍳->>🗄️: SELECT * FROM products WHERE...
    🗄️-->>🍳: 15条结果
    🍳->>⚡: 存缓存（下次直接取）
    🍳-->>🚪: JSON数据
    🚪-->>🧑: 15条商品

    Note over 🧑,📦: 场景三：发聊天消息"多少钱？"
    🧑->>🚪: WebSocket连接
    🚪->>🍳: /ws/chat/xxx/
    🍳->>🗄️: 保存消息
    🍳->>⚡: 广播：对方有新消息
    ⚡-->>🍳: 推送给对方
    🍳-->>🧑: 对方立刻看到

    Note over 🧑,📦: 场景四：上传商品图片
    🧑->>🚪: 上传图片
    🚪->>🍳: /api/products/ (multipart)
    🍳->>📦: 存图片文件
    📦-->>🍳: /media/products/abc.jpg
    🍳->>🗄️: 存商品记录（含图片路径）
    🍳-->>🧑: 发布成功
```
