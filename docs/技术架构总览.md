■ 关键技术选型与理由
  • Vue 3 Composition API — 组合式API，逻辑复用性强，<script setup> 语法简洁
  • Django REST Framework — 成熟稳定，ORM强大，ViewSet+Serializer 开发效率高
  • JWT 双Token机制 — Access(2h) + Refresh(7d)，401自动无感刷新
  • WebSocket (Django Channels) — 实时聊天，Redis Channel Layer 跨进程消息广播
  • MySQL 8.0 — 团队均有使用经验，utf8mb4，Django ORM 适配完善
  • Redis 7 — Channels 必选组件，同时用作热点数据缓存，读写 <1ms
  • MinIO — Docker 本地部署零成本，图片与业务数据库分离
  • Element Plus — 国内最成熟 Vue 3 组件库，风格统一，文档齐全
  • Nginx — 反向代理 + 静态资源直出 + HTTPS，单机支撑 500 并发
